puush-Viewer
============

It's a simple Viewer for the image sharer "puush!". It's badly coded and I guess I won't update it.
Feel free to do whatever you want. 

Controls
--------

Keyboard
W/UP    = Copy link.
A/LEFT  = Go to your last picture.
S/RIGHT = Get a new one.
D/DOWN  = Save image.

Mouse
Left    = Copy link.
Right   = Save image.
